# tomycrypto
TomyCrypto 🔐
A simple yet powerful Flask web application that lets you encrypt and decrypt messages using AES encryption.
Features include:

User login system 🔑 for secure access.

AES Encryption & Decryption 🛡 to protect your data.

History page 📜 to track previous encryptions and decryptions.

Clean UI 🎨 with custom branding and background images.

Built with:

Python (Flask Framework)

HTML, CSS, JavaScript for frontend

SQLite for storing data locally
